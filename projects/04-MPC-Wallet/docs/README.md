# Self Wallet 项目资料
   Self Wallet 是基于 MPC 技术实现的钱包方案，使用三方㧌管实现分布式保护用户私钥安全。绑定邮箱和第三方验证方式，实现私钥随时找回。支持 EOA 和 Eip-4337，ERC20等操作。
独有的功能 4337合约交互方式。让用户免gas费实现转帐。
